# zed-alpha
A first person zombie survival game built in Roblox.

This project focuses on core gameplay systems including:
- Raycast based weapon system
- Zombie AI with pathfinding
- Combat and damage systems

## Features
- First person combat
- Modular weapon system
- Zombie AI (chase + attack)
- Dynamic environment exploration

## What I Learned
- Client->server communication (RemoteEvents / Functions)
- Raycasting for weapon systems
- AI behavior and pathfinding
- Structuring modular systems
